## npm
npm 全称 **Node Package Manager**, 即 Node 的包管理工具。
npm 是 node.js 官方内置的包管理工具

### npm 的安装
node.js 安装时会自动安装 npm，所以如果你已经安装了 node.js，就可以直接使用 npm。
可以通过 `npm -v` 查看版本号测试，如果显示版本号说明安装成功，反之安装失败。
